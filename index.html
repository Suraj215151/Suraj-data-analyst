Project Portfolio :

A Comprehensive Study of Capgemini Employee Reviews

This project aims to analyze employee reviews to uncover trends and factors influencing satisfaction. Key objectives include identifying patterns in satisfaction levels, developing an NLP model for keyword extraction, and creating a recommendation system for tailored suggestions. Additionally, a multiclass sentiment classification model will be built to categorize employee feedback into positive, negative, and neutral sentiments, enhancing insights into employee experiences and guiding improvements.

Time Series and Machine Learning Approach for Weather Forecasting

The project focuses on analyzing and forecasting Pune city's temperature using time series modeling and machine learning techniques. The objectives include studying trend and seasonal patterns in the weather dataset, fitting suitable time series models, and generating forecasts based on trend and seasonality. Additionally, it involves comparing the performance of various time series models to identify the most accurate method for temperature prediction.
